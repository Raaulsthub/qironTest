# How to deploy a AWS LEX bot

### 1. Login to the AWS CONSOLE at https://aws.amazon.com/pt/console/
  ![image](https://user-images.githubusercontent.com/85199336/198062761-0df08ebf-9e2a-40f7-ae51-bdfd11d6dcb3.png)
### 2. Select you Bot in the initial page
  ![image](https://user-images.githubusercontent.com/85199336/198062974-704dd8d1-f751-40ee-b375-2c52689c172e.png)
### 3. In the Bot main page you will find the "Create version and aliases for deployment" topic, press the "View Aliases" option
  ![image](https://user-images.githubusercontent.com/85199336/198063373-e286c59a-cc8a-42a4-b819-e429a83517df.png)
### 4. Press the "Create Aliases" button
  ![image](https://user-images.githubusercontent.com/85199336/198063591-fe15995c-c7e3-4b03-b275-fb15a8a3d5ca.png)
### 5. Fill in the information needed, then press the create button
  ![image](https://user-images.githubusercontent.com/85199336/198063887-4e395d1a-f869-4fa9-8785-ac697de70dd9.png)
### 6. You've deployed the bot, but it's not over, press the link with the name of your deployment
  ![image](https://user-images.githubusercontent.com/85199336/198064093-834d90e7-9546-47c2-9612-2434bcf83b89.png)
### 7. Press the "Associate version with alias" button, and select the version of your bot this release should containm you might want that to be the "Draft Version"
  ![image](https://user-images.githubusercontent.com/85199336/198064993-41510417-f81f-40c2-999b-8c2584da87cd.png)
### 8. If there is no versions apearing to you, that's because the bot don't have any. We need to create one. In the left tab, go to  "Bot versions"
  ![image](https://user-images.githubusercontent.com/85199336/198065829-f0a840eb-d80f-4d7e-830c-932941fd77bb.png)
### 9. Select "Create version"
  ![image](https://user-images.githubusercontent.com/85199336/198065977-264e1249-5604-4fc0-821a-b4ac1182e493.png)
### 10. Provide a description for the version and press "Create"
  ![image](https://user-images.githubusercontent.com/85199336/198066382-286af26a-faca-4d44-b2ac-62871378c9c1.png)
  ![image](https://user-images.githubusercontent.com/85199336/198066470-647b907e-c0e0-4ec6-b476-35644c7dd6e9.png)
### 11. You've created a new version, now just go to the bot alias and link it
  ![image](https://user-images.githubusercontent.com/85199336/198066859-74c70aed-d2b4-46d5-8206-c011f1b00f90.png)
